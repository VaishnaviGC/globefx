# 🌍 GlobeFX — Global Currency Explorer

A real-time interactive currency explorer built with Python and Streamlit.
Convert any world currency to Indian Rupee (INR) instantly!

## 🔗 Live Demo
👉 [globefx.streamlit.app](https://globefx.streamlit.app)

## ✨ Features
- 🗺️ Interactive world map with live currency data
- 🔍 Search any country instantly
- 💱 Real-time currency converter (INR ↔ any currency)
- 📊 Popular currencies quick view
- 📋 Full sortable currency rate table
- 🟢 Live rates updated every 5 minutes

## 🛠️ Built With
- Python
- Streamlit
- Plotly
- Pandas
- Exchange Rate API

## 🚀 Run Locally
1. Clone the repo
   git clone https://github.com/VaishnaviGC/globefx.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py

4. Open browser at http://localhost:8501
